0.- Mostrado de los Script.
Enunciado Ejecutar PDO Mostrar código Ejecutar MySQL Mostrar código
1.- Conexión a la base de datos con la cuenta usuario y tratamiento de errores.
2.- Mostrar el contenido de la tabla Despartamento y el número de registro.
3.- Formulario para añadir un departamento a la tabla Departamento con validación de entrada y control de errores.
4.- Formulario de búsqueda de departamentos por decripción (por una parte del campò DescDepartamento).
5.- Añadimos tres registros a nuestra tabla Departamento utilizando tres instrucciones insert y una transacción, se añadan los tres registros o no se añada ninguno.
6.- Pagina web que cargue registros en la tabla Departamento desde un array departamentosnuevos utilizando una consulta preparada.
7.- Página web que toma datos de un fichero xml y los añade a la tabla Departamento de nuestra base de datos. (IMPORTAR).
8.- Página web que toma datos de la tabla Departamento y guarda en un fichero departamento.xml. (COPIA DE SEGURIDAD / EXPORTAR).
8.- Página web que toma datos de la tabla Departamento y guarda en un fichero en local. (COPIA DE SEGURIDAD / EXPORTAR).
9.- Aplicación resumen MtoDeDepartamentos. (Incluir PHPDoc y versionado en el repositorio GIT).
